What are my Skills?
My skills include JavaScript, Python, HTML and CSS.

What is Python?
Python is a programming language that can be used for a large variety of applications.
What is JavaScript?
JavaScript is a programming language which allows the developer to implement complex features on websites. When a web page isn't static; such as using animated 2D or 3D graphics, scrolling, etc, you will know that it is JavaScript.
What is HTML?
HTML Stands for Hypertext Markup Language. HTML is a markup language which we use to give structure and meaning to our web content. We can define paragraphs, headings, images, etc.
What is CSS?
CSS stands for Cascading Style Sheets. It is a language of style rules which is applied to our HTML content on our websites.
